## Getting Started

1. Make a bot using BotFather in telegram.
2. Enter your bot token and your id in main.py code.
3. Run main.py

### Installing
```
pip install bs4
pip install schedule
pip install python-telegram-bot
pip install telegram
pip install asyncio
```
